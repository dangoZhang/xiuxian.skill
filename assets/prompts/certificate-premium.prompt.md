# AI 协作能力证书生成 Prompt

用于生成「AI 协作能力证书」分享图。目标是简洁、高级、强信息密度，适合社交传播封面。

## 变量

- `{{title}}`：AI 协作能力证书
- `{{username}}`：用户名
- `{{generated_at}}`：生成时间
- `{{level}}`：证书等级，如 L4、L7、L10
- `{{ability}}`：一句能力描述，如“重复跑通常见 workflow”
- `{{token_text}}`：一句 token 文案，如“近 30 日消耗 538,620 token”
- `{{theme_name}}`：主题色名，如 blue、gold、red
- `{{accent_color}}`：强调色，如 #7EAEFF
- `{{panel_color}}`：主面板深色，如 #162B49

## Prompt

请生成一张 3:4 竖版中文证书海报，主题为“AI 协作能力证书”。整体观感需要像高质量、可分享、可晒图的等级认证卡，简洁、克制、专业，不要花哨。

整体要求：

1. 标题必须是“{{title}}”，位于顶部中央，中文排版挺拔、干净。
2. 标题下方需要有两条并列信息：
   - 持有人：{{username}}
   - 生成时间：{{generated_at}}
3. 画面中央是主证书面板，核心只突出三层信息：
   - 超大等级：{{level}}
   - 小字标签：CURRENT LEVEL
   - 能力描述：{{ability}}
4. 主面板底部再放一行 token 信息：{{token_text}}。
5. 保持留白，不再额外塞入次级解释区块。

视觉要求：

- 采用现代高级认证证书风格，背景简洁，主体卡片突出。
- 主色使用 {{theme_name}} 对应的高级主题色，强调色使用 {{accent_color}}，主面板用 {{panel_color}}。
- 可以有轻微光晕、金属边框、浅浮雕线条，但只能增强品质感。
- 文字排版必须整齐、统一、对齐，不能出现字号忽大忽小和块距混乱。
- 等级必须一眼可见，能力描述必须简短有力。

禁止项：

- 不要修仙、玄幻、符箓、火焰、人物立绘。
- 不要加入多余图标、奖杯、勋章、AI 机器人头像。
- 不要塞入过多说明文字。
- 不要让卡片看起来像 PPT、后台面板或简历模板。
